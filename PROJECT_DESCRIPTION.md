# 🏰 Throne and Liberty Clan Website имя клана Eternal

## 📋 Обзор проекта

Веб-сайт для клана в игре **Throne and Liberty** - это комплексная платформа для управления кланом, коммуникации между участниками и привлечения новых игроков. Сайт объединяет функциональность социальной сети, информационного портала и системы управления кланом.

---

## 🎯 Цели проекта

### Основные цели:
- **Создание централизованной платформы** для всех участников клана
- **Упрощение коммуникации** между членами клана
- **Привлечение новых игроков** в клан
- **Информирование** о новостях игры и обновлениях
- **Организация игровых событий** и активности клана

### Целевая аудитория:
- **Текущие члены клана** - для координации и коммуникации
- **Потенциальные новички** - для ознакомления с кланом
- **Альянсы и партнеры** - для межкланового взаимодействия

---

## 🏗️ Архитектура сайта

### Структура разделов:

#### 1. **Главная страница** (`/`)
- Приветствие и краткая информация о клане
- Последние новости и события
- Статистика клана (количество участников, уровень клана)
- Быстрые ссылки на основные разделы
- Форма быстрой регистрации для новичков

#### 2. **О клане** (`/about/`)
- История и философия клана
- Цели и миссия
- Правила клана
- Структура управления
- Достижения клана
- Скриншоты и видео

#### 3. **Новости** (`/news/`)
- Новости игры Throne and Liberty
- Обновления и патчи
- Новости клана
- Анонсы событий
- Категории новостей (Игра, Клан, События)
- Поиск и фильтрация

#### 4. **Участники клана** (`/members/`)
- Список всех участников
- Профили игроков с детальной информацией
- Роли и звания в клане
- Статистика активности
- Онлайн статус игроков
- Система рейтинга

#### 5. **События и активность** (`/events/`)
- Календарь событий клана
- Планирование рейдов и PvP
- Регистрация на события
- Отчеты о прошедших событиях
- Система напоминаний

#### 6. **Форум** (`/forum/`)
- Разделы для обсуждений
- Темы по игре и клану
- Система комментариев
- Поиск по темам
- Модерация контента

#### 7. **Галерея** (`/gallery/`)
- Скриншоты из игры
- Видео с событий
- Достижения клана
- Мемы и развлекательный контент

#### 8. **Контакты** (`/contacts/`)
- Контактная информация лидеров
- Discord сервер
- Социальные сети
- Форма обратной связи

## 🔧 Техническая документация

### Структура базы данных:
```
Users (Пользователи)
├── UserProfile (Профили)
├── ClanRole (Роли в клане)
└── UserActivity (Активность)

News (Новости)
├── NewsCategory (Категории)
└── NewsTag (Теги)

Forum (Форум)
├── ForumCategory (Разделы)
├── ForumTopic (Темы)
└── ForumPost (Сообщения)

Events (События)
├── EventCategory (Типы событий)
└── EventRegistration (Регистрация)

Gallery (Галерея)
├── Album (Альбомы)
└── Media (Медиафайлы)
```

### API Endpoints:
```
/api/v1/
├── auth/ (Аутентификация)
├── users/ (Пользователи)
├── news/ (Новости)
├── forum/ (Форум)
├── events/ (События)
├── gallery/ (Галерея)
└── clan/ (Клан)
```
