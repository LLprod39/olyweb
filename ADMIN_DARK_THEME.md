# Темная тема для админ панели OlyWeb

## Описание

Современная темная тема для админ панели Django с красивым дизайном и удобным интерфейсом. Тема создана специально для сайта клана Throne and Liberty (Eternal).

## Особенности

### 🎨 Цветовая схема
- **Основной фон**: `#0f1419` (темно-синий)
- **Поверхности**: `#1a1f2e` (темно-серый)
- **Карточки**: `#242938` (серый)
- **Границы**: `#2d3748` (светло-серый)
- **Текст**: `#e2e8f0` (белый)
- **Акцент**: `#6366f1` (индиго)

### ✨ Современный дизайн
- Плавные анимации и переходы
- Градиентные кнопки и элементы
- Скругленные углы (8-12px)
- Тени и эффекты hover
- Адаптивный дизайн

### 🚀 Интерактивность
- Анимации при наведении
- Плавные переходы
- Индикаторы загрузки
- Улучшенная навигация

## Файлы

### CSS файлы
- `static/css/admin-custom.css` - Основные стили темной темы
- `static/css/admin-dark-override.css` - Переопределение стилей Django Admin
- `static/css/admin-force-dark.css` - Принудительное переопределение всех элементов

### JavaScript файлы
- `static/js/admin-custom.js` - Интерактивность и анимации

### Шаблоны
- `templates/admin/index.html` - Главная страница админки
- `templates/admin/base_site.html` - Базовый шаблон

## Установка

1. Убедитесь, что все файлы находятся в правильных директориях
2. Перезапустите сервер Django
3. Очистите кэш браузера

## Совместимость

- Django 3.2+
- Django Admin
- Jazzmin (если используется)
- Современные браузеры

## Настройка

### Изменение цветов
Отредактируйте CSS переменные в начале файла `admin-custom.css`:

```css
:root {
    --dark-bg: #0f1419;
    --dark-surface: #1a1f2e;
    --dark-card: #242938;
    --primary: #6366f1;
    /* ... */
}
```

### Добавление новых стилей
Создайте дополнительные CSS файлы и подключите их в шаблонах:

```html
{% block extrastyle %}
    {{ block.super }}
    <link rel="stylesheet" href="{% static 'css/your-custom.css' %}">
{% endblock %}
```
